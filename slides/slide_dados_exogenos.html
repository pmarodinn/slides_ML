A Importância dos Dados Exógenos

O que são Dados Exógenos?

Dados externos ao sistema tradicional de crédito que fornecem contexto adicional e aprofundado sobre o comportamento financeiro e perfil do cliente.

Um banco de dados robusto e diversificado é fundamental para alimentar modelos de ML eficazes:
Dados Internos Tradicionais

Histórico de pagamentos, valor do empréstimo, prazo, renda declarada.

Dados Socioeconômicos

Indicadores econômicos (inflação), taxa de desemprego regional, sazonalidade do setor de atuação.

Dados Comportamentais

Padrões de consumo, histórico de navegação em plataformas financeiras, interações com serviços.

Dados Alternativos

Pagamentos de serviços públicos (água, luz), histórico de aluguel, comportamento em redes sociais (se relevante e ético).
